# SignalMatch
SignalMatch is an IR recognition device that allows users to control other devices from their remote signals. When a remote sends a signal to the device, it captures it and matches it with a database of known IR signals. If multiple matches are found, the user selects the correct device from a list of possibilities. Once identified, SignalMatch provides a full set of controls for that device, essentially turning it into a universal remote.
